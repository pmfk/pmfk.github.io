# 最近读的书

# 张力与界限

像这个时代所有的互联网创业者一样，中共在构建完模式闭环后，开始迫不及待地推进规模化。而这个模式也不负众望，显示出它的强大的能量，迅速催生出中央苏区以及一大批革命根据地，分部在华夏各处。

中共在国民党清理后，于生死攸关之际，奇迹般走通了一条具有相当可行性的模式，即农，田，兵履带式扩张的模式。

将土豪的田地以及宗族公田分给贫农，获得对革命的拥戴；

从获得土地的农民群体中征兵，壮大革命有生力量；

以军队暴力机器为后盾，打碎农村传统权力结块，重组乡村生产结构，划分生产资源。（其中重要一环就是打土豪）

这样一个以农田兵三要素为改造对象的模式，形成一个三角，简洁坚固，不可不谓中共在早期创业时的一个创举。

像这个时代所有的互联网创业者一样，中共在构建完模式闭环后，开始迫不及待地推进规模化。而这个模式也不负众望，显示出它的强大的能量，迅速催生出中央苏区以及一大批革命根据地，分部在华夏各处。

然而，在国民党愈渐强大的围剿压力下，那铁一般的三角，似乎也开始被腐蚀，变形，甚至有崩裂的预兆。农，田，兵三个轮子摇摇晃晃，再也无法推动中共战车齐驱向前。

天佑中共，在国共同质竞争之际，一位异质对手，带着颠覆整个市场，大区共荣的野望，粉墨登场。而中共也在命悬一线时成功抓住这个机会，转型成功，并继而摧枯拉朽般完成对老对手的，最终实现垄断。

苏维埃，是共和国的一个不折不扣的高保真原型。当前成熟产品显现出来的所有弊端，在80多年前的原型中你都能找到。错误从未被修正，版本不断在重复。

推荐你认真读下《张力与限界：中央苏区的革命》这本书，看看产品的原型图，你可能会对产品接下来如何迭代有所感悟。

但看完这本书，我的感觉是，无论你作者再怎么调整措辞搞春秋笔法，某些主题依然不可触碰，这不是射箭射中几环的问题，而是根本就不该被允许瞄准靶子。但吊诡的正是这点，书居然上市了，而且对那些主题的叙述似乎逃脱了天眼，安然无恙到今天。

# bad blood

一本相当有意思的书，讲一个斯坦福辍学生创办的生物科技企业，如何骗过美国政界，商界以及风险投资行业，从崛起到陨落的过程。

伊丽莎白·霍尔莫斯（Elizabeth Holmes）从小就知道自己将会成为亿万富豪。

当它还是扎着羊角辫的小娃娃时，她就一本正经的对家人说，我要成为亿万富翁。他们打趣道，你为啥不想当总统呢？她回答说，不了，当了亿万富翁，总统就会来娶我。

良好的家境和教育，使得伊丽莎白长成一个自信，活力满满，不到目的决不罢休的大姑娘，没有什么事能难倒她。高中时，当斯坦福大学一个暑期中文培训班的负责人告诉她，我们只收大学生，伊丽莎白没用多长时间就说服对方同意自己加入。

伊丽莎白从小就开始学中文。因为她爸去过中国，相信21世纪是属于中国人的。十多年后，当她穿着标志性的黑色高领衫，同马云一起登上克林顿全球倡议，接受比尔·克林顿采访时，我们无从知道，她是否在台下用一口流利的中文让马云惊讶。

但有一点可以肯定，斯坦福的那个暑期中文培训项目，相当程度的改变了她的一生。正是在那里，她遇见了自己未来的男朋友，一个大她将近20岁的印度人。

2002年春，伊丽莎白成功考入斯坦福，学习化学工程。同大多数刚进大学的平常人一样，上课，搞派对，谈恋爱。此外，她又一次用自己的口才，说服了学院院长罗伯逊（Channing Robertson），加入了他的实验室当助手。

在第一个学期快要结束时，她猛然醒悟，不能再这样优哉游哉过日子。老娘要赶紧实现儿时的梦想——成为亿万富翁！刚刚入学几个月，伊丽莎白就已经产生了退学的念头。她走进男朋友的宿舍，用坚定的语气告诉对方，分手吧，老娘要去开公司，没工夫谈恋爱。说完便头也不回离开，留下对方一脸懵逼。

2003年夏，非典肆虐亚洲，人心惶惶。伊丽莎白在新加坡参加一个医疗项目的实习，帮忙测量血液中的病毒。陈旧落后医疗器材和技术，让她深感革新的需要。回到美国的家中，她立即坐到电脑面前，敲敲打打，一连5天，每晚只睡一两个小时。当她妈妈开车送她去学校读大二时，伊丽莎白在后座睡得香甜，手边放着一份专利申请书。

一到学校，她就冲进实验室，把她的专利申请给罗伯逊院长看。罗伯逊看完，掩饰不了内心的激动，连连称赞伊丽莎白的天才想法，将科学，工程，技术三者结合，堪称颠覆！像你这样的人才，我还是第一次见！

罗伯逊用颤抖的手把专利递给他另一位助手肖纳克（Shaunak），肖纳克翻开一看，伊丽莎白设想的是一个手臂帖（arm patch），贴在手臂上，能同时检测和治疗疾病。肖纳克暗想，这姑娘是不是科幻片看多了？但连见多识广的院长都如此激动，那这个东西肯定是可行的，肖纳克打消了疑虑。

得到二人的肯定后，伊丽莎白开始准备申请专利的文书。

2004年5月，肖纳克加入伊丽莎白的公司，成为其第一号员工，拥有少量股权。院长罗伯逊则担任公司顾问。

伊丽莎白一开始将公司命名为「Real Time Cures」，意思是「实时治疗」，但「Cures」拼写起来太像「Curse」也就是「诅咒」，她就换了个新名字「Theranos」，这个单词由「Therapy」，意为「疗法」，和「Diagnosis」，意为「诊断」，两个词组合而成。

创业当然离不开资金。伊丽莎白第一个想到了自己儿时的邻居的父亲，蒂姆（Tim Draper），蒂姆这个人可不简单。他爷爷从1950年代起就在硅谷成立了第一家风险投资机构，是风投业的先驱。

蒂姆自己也创办了一家风投机构，DFJ（德丰杰）。德丰杰投资的成功案例包括Hotmail，Skype，特斯拉，Twitter等等，在中国，他投资了百度和分众传媒等著名公司。

看见邻居来融资，能拒绝吗？德丰杰二话不说，立刻投了她100万美元。

之后，她又通过家族人脉以及斯坦福校友，又成功融了大约500万美元。投资者中有做房地产的，搞私募股权的，还有一个掌握台湾医疗器械渠道的台湾人，伊丽莎白妈妈的姊妹也投了点。

不过融资过程也并非畅通无阻。在见一个名为「MedVenture Associates」的投资机构时，伊丽莎白被对方咄咄逼人地询问产品细节。他们质问她，你的产品和现在市面上的「Abaxis」这个产品有什么区别吗？伊丽莎白答不出来，其它关于自己产品的细节，她也说不清楚。

那几个投资人以为伊丽莎白不过是头脑发热，一个想法改变世界？这样的人他们见得太多了。会谈了1个小时后，伊丽莎白满脸通红离开离开了，两手空空。

截止2004年12月的第一轮融资，总共大约600万美元。伊丽莎拿钱白招兵买马，项目很快就启动了。

没过多久，她开始觉得，自己的点子，行不通。这不是废话吗，小学生都能看出来，拿一块手臂贴来诊断和治疗疾病，科幻片看多了吗？你为何不上天和太阳肩并肩？！

她做出调整，不再要求手臂贴有治疗功效，能诊断出疾病就可以了。但是，这个目标依然非常难达到。

她没有那么容易放弃。最后，伊丽莎白苦思冥想，想出来个新玩意儿。按照她的设想，新设备由两部分组成。一个厚厚的信用卡般大小的匣子，以及一个读数器（reader），病人先用针轻轻刺破指尖，挤挤，取一滴血，然后把它放入匣子，匣子再放进读取器，读取器进行一系列的化验操作，最后把化验结果显示在读取器的液晶屏幕上。

伊丽莎白希望匣子能尽可能小，最好能手持。读取器能够放在病人家里，当病人独立完成检测后，读取器将数据传输到Theranos公司的中央服务器上，接着再被转发给医生。这样病人能够和医生进行实时沟通，不用等到下一次会面时才能知道自己的健康状况。

最重要的一点，伊丽莎白希望Theranos的血液检测，不仅能够测量血糖，还能够检测血液中大多数元素。听起来比手臂帖靠谱一些，不过说实话，依然有点科幻色彩。

2005年年末，Theranos公司的第一个原型设备成功开发出来！！！公司此时已有24位员工。商业模式也已经确立，即和制药公司达成协作。

制药公司每年要花几十上百亿美元投入在药物研发上，但绝大多数的药物实验都是失败的，因此会浪费掉大量财物。如果Theranos和制药公司合作，为他们提供自己的技术，制药企业就不必等到研发末期才能知道药物效果，他们能在药物实验的早期就通过实时监测病人的血液，拿到数据，做出是药物是否有效的判断。如果有效，就继续投入资金展开后续研发；如果无效，就及时止损，终止该项药物的研发。用这样的方法，制药公司能省下几十亿美元。

不得不说，这个商业模式相当靠谱。公司管理层和投资人都对Theranos的未来非常看好，员工也非常有激情。

Theranos慢慢开始获得公众关注，一家叫《红鲱鱼》的杂志对她伊丽莎白进行了采访报道，称Theranos「硅谷最热门公司」。

一切看起来都很完美，伊丽莎白成为亿万富豪的梦想，近在咫尺。她只要把产品搞定，推向市场，无数的财富和荣耀，会像潮水一样将她包围。唯一要做的，就是把产品尽快制造出来！

伊丽莎白坐在办公室，员工早已下班回家，办公室只有她一人。公司的美好未来，以及浓咖啡，让她的神经极度亢奋。在网络上搜索一番后，她的目光落在一个人的名字上，他能帮公司搞定产品研发：艾德（Edmond Ku）

艾德是香港人，说英文时只会用一般现在时态。他善于解决问题，在硅谷有口皆碑。2006年年初，伊丽莎白和艾德进行了一次面谈。

伊丽莎白邀请他加入Theranos，把公司的产品原型 Theranos 1.0，打造成为一个可用的产品。她告诉艾德，Theranos不仅仅有广阔的商业前景，更重要的是，它能够真正做到改变世界，拯救无数人的生命。

艾德被伊丽莎白的理想主义感染，加入了Theranos。没过多久，艾德就开始后悔当初的决定。公司的原型实在是太多坑，完全用不了，根本不是伊丽莎白当初跟自己说的那样。另外，伊丽莎白对产品细节非常苛刻，必须做到只需要少量的指尖血就能检测疾病。

艾德问她，为什么要纠结于血量，抽胳膊的血不行吗。伊丽莎白回答说，「我怕针。她妈妈也是，一看见注射器就会晕」血量太少，很多测量无法进行，但伊丽莎白依然不退让。

艾德只能对血液进行稀释，但这样一来，测量结果的差异就变得相当大，

她还要求匣子要做得尽可能小，能放在手里，就像iPod一样。这个对艾德来说难度也很大。至于读取器，那就更难做了，内部构造太复杂，牵一发而动全身，稍微不注意设备就用不了。

好在艾德还算很敬业的，虽然工程技术上的问题很多，他还是硬着头皮继续干了。但他始终认为，如果能利用更多的血液，问题更容易解决，但伊丽莎白无论如何都不妥协。

而最令艾德不舒服的，不是技术问题，而是团队沟通。公司禁止不同团队之间相互沟通。这个规定让艾德很无语，相互之间无法直接沟通，那我怎么知道产品问题是出在我负责的工程技术上，还是出在化学团队那边？？？

除了伊丽莎白，谁都不知道产品的全貌。

艾德很郁闷，不过没办法啊，来都来了，继续干呗。

公司还是有前景的。艾德在来到公司不久，就发现伊丽莎白的办公室里挂着一条语录「你开始意识到你正凝视着下一个乔布斯或比尔盖茨的眼睛」。这句话出自罗伯逊院长，他一直对伊丽莎白青睐有加。

随着产品研发，公司的开销开始增大。匣子需要从公司外部订购，1个匣子，花费200多美元，而且只能用一次。公司每周要测试几百个匣子。

第一轮融资来的600万美元，很快就烧完了。不过没关系，伊丽莎白很快又融到了900万美元。

卢卡斯（Don Lucas）和甲骨文公司创始人——埃里森（Larry Ellison）两位大佬级人物加入了第二轮融资。埃里森经常开着他的红色保时捷来公司看看。

作为创始人和CEO，伊丽莎白工作起来简直不要命，每天只睡4个小时，依靠大量咖啡豆来保持清醒。艾德劝她别透支身体健康，她拒绝了。

伊丽莎白不仅自己玩命工作，还让员工也跟她一样。当艾德的工作没什么进展时，她甚至要求工程团队一天24个小时，每周7天，日夜不休地加快开发。

艾德心想，老板你还是不是人。其实他的团队已经很努力加班了。艾德拒绝了伊丽莎白，并且告诉她，那样会让工程师们崩溃的。伊丽莎白说：我不在乎他们，我只在乎公司。

自从艾德拒绝了自己后，伊丽莎白和他的关系迅速冷却到冰点。

而这个时候，艾德发现，公司的 CFO 不知道什么时候从公司消失了。
